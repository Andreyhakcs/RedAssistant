
One‑click portable build — FIXED spec (без требования папки config)

Шаги:
1) Распакуй содержимое архива в корень проекта (рядом с папкой red2/).
2) .env уже внутри архива. При необходимости отредактируй ключ в .env.
3) Запусти Make-App-Portable.bat — он:
   - почистит build/dist
   - установит PyInstaller (если нужно)
   - соберёт папку dist/RedAssistant/
   - скопирует .env в папку сборки
   - запустит RedAssistant.exe

Если снова будет ошибка — пришли скрин, глянем.
