
Patch: TTS Voice в Settings
---------------------------
Что даёт:
- Появился выпадающий список голосов (через pyttsx3) + Test TTS.
- Выбранный голос сохраняется и применяется без перезапуска.
- Совместимость: если твой класс TTS не принимает параметр 'voice', пробуется метод set_voice().

Установка:
1) Распакуй в корень проекта с заменой файлов:
   - red2/ui/user_prefs.py
   - red2/ui/settings_dialog.py
2) Открой свой red2/app.py и вставь хелпер из файла:
   - red2/app_voice_snippet.txt
   В двух местах используй `_make_tts_from_prefs(self.prefs)` вместо прямого TTS(...):
   а) при первом создании `self.tts = ...`
   б) в apply_prefs() при пересоздании TTS
3) Перезапусти. В Settings появится «TTS Voice».

Примечания:
- На Windows 10 голоса берутся из установленных «Речь» (Параметры → Время и язык → Речь → Добавить голоса).
- Если список пустой — установи дополнительные голоса системы.
